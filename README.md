A basic command-line expense tracker built using Python and CSV.

## Features

* Add new expenses (Item name + Price)
* View all saved expenses
* Stores data in a CSV file (`expense.csv`)
* Handles invalid inputs safely

## How It Works

The program gives three options:

1. Add Expense
2. View Expense
3. Exit

All expenses are saved permanently in a CSV file.


## How to Run

```bash
python filename.py
```
